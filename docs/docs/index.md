# CALFEM for Python

Welcome to the documentation for CALFEM for Python. On this page you
will find Python-specific installation notes, mesh generation guidance,
examples, and reference documentation for the modules contained in the
CALFEM for Python distribution.

The [CALFEM manual](https://calfem-python-manual.readthedocs.io/en/latest/)
is the main source for the finite element method background, standard
function groups, and canonical examples. Use these pages for
Python-specific workflows that are not covered there, especially the
mesh generation guide and API reference.
